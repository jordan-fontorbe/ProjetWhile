Remarques : 
Le Code 3 adresse est fourni pour l'ensemble du programme.
Les erreurs remarquées à la compilation qui donnent lieu à des sorties sur yyerror ne sont pas bloquantes. 
Elles s'affichents avant les tables de symboles

Les variables temporaires n'aparaissent pas dans la Table des symboles.
Les etiquettes d'expression conditionelles n'apparaissent pas dans la Table des symboles.


L'opérateur OP_EXP n'est pas géré lorsqu'utilisé avec des variables déclarées calculées(non calculées).
Il n'y a pas de vérification du type des arguments lors d'un appel d'une fonction ou d'une procédure,
seul le nombre d'argument est comparé.


Le Code d'instruction For n'est pas utilisable.
Il n'y a pas d'optimisation au niveau des expressions et des blocs.

Il y a un dossier : ftests qui contient divers fichiers de test.
